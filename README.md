## Getting Started

### Prerequisites

This is the list of things you need to use and how to install them.
    <br />

* npm
    <br />

  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
    <br />

   ```sh
   git clone https://github.com/fatemehMirabdollahi/to-do.git
   ```
2. Install NPM packages
    <br />

   ```sh
   npm install
   ```
